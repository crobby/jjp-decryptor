# Pirates of the Caribbean — Audio File Guide

A reference for modders who want to customize audio in JJP's Pirates of the Caribbean pinball.

**Total: 2,338 audio files (645 MB)**

## Modding Tips

1. **Start with music** — the 52 OGG files in `sound/music/` give the biggest bang for the buck. Replacing wizard mode or chapter themes completely changes the vibe.

2. **Match duration** — the game engine expects audio files to be close to the original duration. The JJP decryptor tool handles WAV trim/pad automatically; OGG files are resized via ffmpeg.

3. **Keep the format** — WAV files must stay WAV, OGG files must stay OGG. Same sample rate and channel count as the original.

4. **Numbered variants** — files like `FX_BumperRubber_001–004` are random variations the game picks from. You can replace all of them or just some.

5. **`_Null_` files** — these are intentionally silent. Don't replace them with audible sounds unless you want to add audio where the game is intentionally quiet.

6. **Gibbs speech** — there are 1,549 Gibbs voice lines. Replacing all of them is a massive project, but you could target specific categories (e.g., just the 45 Liar's Dice lines or the 41 instruction lines).

7. **Fanfares are impactful** — the `Fanfare_*` files play during jackpots, mode completions, and achievements. Swapping these gives immediate feedback during exciting moments.
